# Viral_discovery_pipeline_museum
A pipeline for characterising viral fragments from museum materials (formalin- and ethanol-fixed wildlife specimens) using metatranscriptomics. Designed for finding fragmented viral genomes.


Museum_viral_pipeline.pbs was used in manuscript "Novel 20th century rotavirus alphagastroenteritidis strains identified in formalin- and ethanol-fixed museum specimens" published in MBE. This pipeline was used to characterise novel rotavirus lineages from museum specimens. Other scripts referenced in this manuscript are kalign_mapping.pbs and STAR_mapping.pbs.

The most recent version of the pipeline is hanta_QC_assembly_blast_pipeline.pbs and trinity_array.sh (March 2026), along with scripts contig_mapping.pbs for contig mapping (estimating abundance of viral contigs in sequencing libraries) and ML_trees.sh (viral alignments and ML trees).
